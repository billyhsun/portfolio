Boxhead Video Game

ICS3U - Computer Programming, Earl Haig Secondary School


This is the final project for the grade 11 computer programming course, which involved the design and building of a video game using Python and the Python library Pygame.

This game is an attempt to recreate the well-known online video game Boxhead. For more information, please see the presentation.
